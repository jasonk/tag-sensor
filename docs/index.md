## Configuration ##

To use the tag-sensor tool, you'll need a configuration file. You can
check out these config files from the repository for a good starting
point:

* [example-config.yaml](https://github.com/jasonk/tag-sensor/blob/main/example-config.yaml)
* [test-config.yaml](https://github.com/jasonk/tag-sensor/blob/main/test-config.yaml)

To get started you need to configure one or more of each of the
following items:

 * [Cameras](./cameras.md)
 * [Markers](./markers.md)
 * [Interfaces](./interfaces.md)
